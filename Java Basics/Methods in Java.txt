Method is a field created by user in Java, it is made to perform some action at specific moment once method is being called.
method can be called at any line and any moment. think of it was storing an action that you can perform in any situation
you want. methods need to have a keyword 'void' to specify that it is a method. 
methods go inside of the class:
public class Main {
    void create1() {
        create2(); //method being called
   }
    void create2() { 
        //code goes here
     }
}
